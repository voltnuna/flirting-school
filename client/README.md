### 📚FILE STRUCTURE

```
- 📂 public
    - 📄 electron.ts : electron config file
    - 📄 preload.js : electron preload
- 📂 src
  - 📂 assets
    - 📂 fonts
    - 📂 images
    - 📂 scss
    - 📂 ts
- 📂 components
- 📂 layouts
- 📂 pages
- 📂 typings
- 📂 utils
- 📂 hooks
  - 📄 useBoolean : 모달 open, hidden을 위한 hooks
  - 📄 useInputs : input 값 handle을 위한 hook

- ✅📄 tsconfig.json : js설정파일 (추가할 경로가 있을 경우 이 파일에서 셋팅한다.)
```
